| 数据源管理 | 管理数据表和字段 | 配置模式 → 数据源 |

| 插件管理 | 安装和管理插件 | 配置模式 → 插管理 |

---

### 2. 数据源管理

#### 什么是数据源？

数据源是 NocoBase 的核心，定义了数据的结构和存储位置。

#### 数据源类型

| 类型 | 说明 | 使用场景 |
|------|------|----------|
| 主数据库 | NocoBase 自带的数据库 | 大部分业务数据 |
| 外部数据库 | 连接外部 MySQL/PostgreSQL | 集成现有系统 |
| API 数据源 | 通过 REST API 获取数据 | 对接第三方系统 |

#### Collection（数据表）

Collection 类似于数据库中的表，包含多个字段。

**创建 Collection：**
1. 进入配置模式
2. 点击「数据源」→「主数据源」
3. 点击「创建集合」
4. 输入集合名称和字段

#### Field（字段）

常用字段类型：

| 类型 | 说明 | 示例 |
|------|------|------|
| 单行文本 | 短文本 | 姓名、标题 |
| 多行文本 | 长文本 | 描述、备注 |
| 数字 | 整数或小数 | 年龄、金额 |
| 日期 | 日期时间 | 创建时间 |
| 单选 | 固定选项 | 状态、优先级 |
| 多选 | 多个选项 | 标签 |
| 关联 | 关联其他表 | 外键 |

---

### 3. 区块类型

区块是展示数据的方式：

| 区块类型 | 说明 | 使用场景 |
|----------|------|----------|
| 表格区块 | 表格形式展示数据 | 数据列表 |
| 表单区块 | 表单形式编辑数据 | 新增/编辑 |
| 详情区块 | 详情页展示 | 查看详情 |
| 看板区块 | 看板视图 | 任务管理 |
| 日历区块 | 日历视图 | 日程安排 |
| 图表区块 | 数据可视化 | 统计分析 |

**添加区块：**
1. 进入配置模式
2. 在页面内容区域点击「添加区块」
3. 选择区块类型
4. 选择数据源
5. 配置字段和操作

---

### 4. 操作类型

操作是对数据的处理动作：

| 操作类型 | 说明 | 使用场景 |
|----------|------|----------|
| 新增数据 | 添加新记录 | 创建按钮 |
| 编辑数据 | 修改记录 | 编辑按钮 |
| 查看详情 | 查看记录详情 | 详情按钮 |
| 删除数据 | 删除记录 | 删除按钮 |
| 筛选 | 筛选数据 | 筛选器 |
| 导入 | 批量导入数据 | 导入按钮 |
| 导出 | 导出数据 | 导出按钮 |
| 工作流 | 触发工作流 | 自定义按钮 |

**添加操作：**
1. 进入配置模式
2. 在区块操作栏点击「配置操作」
3. 选择操作类型
4. 配置操作参数

---

### 5. 实战练习：创建一个简单的任务管理

#### 步骤 1：创建数据表

创建「任务」集合，包含字段：
- 标题（单行文本）
- 描述（多行文本）
- 状态（单选：待办/进行中/已完成）
- 优先级（单选：低/中/高）
- 截止日期（日期）

#### 步骤 2：创建页面

1. 进入配置模式
2. 点击左侧菜单「添加菜单项」
3. 选择「添加页面」
4. 命名为「任务管理」

#### 步骤 3：添加表格区块

1. 在页面中点击「添加区块」
2. 选择「表格区块」
3. 选择「任务」集合
4. 配置显示字段

#### 步骤 4：添加操作

1. 在表格区块配置操作
2. 添加「新增」「编辑」「删除」操作
3. 测试功能
