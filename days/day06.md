# Day 6：界面搭建

> **学习目标：** 掌握页面、菜单、区块的配置方法
> **预计时间：** 1.5 小时
> **难度：** 🟡 进阶

---

## 📖 学习内容

### 1. 页面管理

#### 创建页面

1. 进入配置模式
2. 点击左侧菜单「添加菜单项」
3. 选择「添加页面」
4. 配置页面属性：
   - 页面标题
   - 图标
   - 路由路径

#### 页面类型

| 类型 | 说明 |
|------|------|
| 普通页面 | 自由添加区块 |
| 详情页 | 展示单条记录详情 |
| 弹窗页面 | 以弹窗形式打开 |

#### 页面布局

```
┌─────────────────────────────────────┐
│ 页面标题 │
├─────────────────────────────────────┤
│ 区块 1 │
├─────────────────────────────────────┤
│ 区块 2 │ 区块 3 │
└─────────────────────────────────────┘
```

---

### 2. 菜单配置

#### 菜单层级

```
顶部导航（一级菜单）
├── 左侧菜单组（二级菜单）
│ ├── 页面 A
│ ├── 页面 B
│ └── 链接 C
└── 左侧菜单组 2
└── ...
```

#### 菜单项类型

| 类型 | 说明 |
|------|------|
| 页面 | 打开 NocoBase 页面 |
| 链接 | 打开外部链接 |
| 分组 | 组织菜单层级 |

#### 菜单权限

可以为不同角色配置不同的菜单访问权限。

---

### 3. 表格区块

#### 基本配置

1. 添加区块 → 选择「表格区块」
2. 选择数据源/集合
3. 配置显示字段
4. 配置操作按钮

#### 列配置

- 显示/隐藏列
- 列宽设置
- 排序
- 筛选

#### 行配置

- 行高
- 斑马纹
- 固定列

---

### 4. 表单区块

#### 基本配置

1. 添加区块 → 选择「表单区块」
2. 选择数据源/集合
3. 配置表单字段
4. 配置提交操作

#### 表单布局

- 普通布局（垂直）
- 网格布局（多列）
- 内联布局（水平）

#### 表单验证

- 必填验证
- 格式验证
- 自定义验证规则

---

### 5. 详情区块

用于展示单条记录的详细信息。

#### 配置步骤

1. 添加区块 → 选择「详情区块」
2. 选择数据源/集合
3. 配置显示字段
4. 配置操作按钮

---

### 6. 看板区块

适用于任务管理、项目进度等场景。

#### 配置步骤

1. 添加区块 → 选择「看板区块」
2. 选择数据源/集合
3. 选择分组字段（如状态）
4. 配置卡片显示内容

---

### 7. 实战：搭建工单管理页面

#### 页面 1：工单列表

- 表格区块展示所有工单
- 筛选器：状态、优先级、分类
- 操作：新增、编辑、删除、导出

#### 页面 2：工单详情

- 详情区块展示工单信息
- 操作记录区块
- 相关工单区块
